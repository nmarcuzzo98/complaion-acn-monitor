Come funziona Complaion — ACN Monitor

Questo portale è uno strumento non ufficiale sviluppato da Complaion per supportare le aziende nel monitoraggio degli aggiornamenti normativi pubblicati dall'Agenzia per la Cybersicurezza Nazionale (ACN) in materia di NIS2.

Cosa fa

  • 🔍 Scansiona automaticamente le pagine ufficiali ACN e i PDF correlati a NIS2.
  • 🔐 Calcola un hash SHA256 di ciascuna risorsa per rilevare variazioni millimetriche.
  • 📊 Mostra in dashboard lo stato di ciascuna risorsa (invariata, modificata, nuova).
  • 📜 Mantiene uno storico degli ultimi 180 giorni di variazioni.

Frequenza scansioni

La scansione è eseguita automaticamente ogni ora tramite GitHub Actions. I dati sono pubblicati in un repository GitHub pubblico.
