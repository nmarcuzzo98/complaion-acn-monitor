🛡️ Come funziona

Questo portale è uno strumento non ufficiale sviluppato da Complaion per supportare le aziende nel monitoraggio degli aggiornamenti normativi pubblicati dall'Agenzia per la Cybersicurezza Nazionale (ACN) in materia di NIS2.

Cosa fa

  • 🔍 Scansiona automaticamente le pagine ACN e i PDF correlati alla NIS2.
  • 🔐 Calcola un hash SHA256 di ciascuna risorsa per rilevare variazioni anche minime.
  • 📊 Mostra in dashboard lo stato di ogni risorsa.
  • 📜 Mantiene uno storico degli ultimi 180 giorni di variazioni.
  • 🚀 Link diretto alla risorsa ACN modificata per consultazione immediata.

Frequenza scansioni

Scansione automatica ogni ora via GitHub Actions.
